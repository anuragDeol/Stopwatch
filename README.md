# Stopwatch
A simple stopwatch web application built using HTML, CSS, and JavaScript. You can use it to time anything you want.
## Usage
To use the stopwatch, simply open the index.html file in your web browser. You will see a timer displayed on the page with a ```Start``` button and a ```Reset``` button. Clicking the "Start" button will start the timer, and clicking it again will pause it. Clicking the "Reset" button will reset the timer to 00:00:00.
